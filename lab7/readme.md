# Лабораторна робота №7: Комп'ютерний зір та детекція об'єктів

Цей проєкт реалізує конвеєр обробки відеопотоку з використанням бібліотеки OpenCV. Метою роботи було впровадження модуля детекції об'єктів на базі нейронної мережі MobileNet-SSD.

## Основний функціонал
- **Фільтрація зображень:** Реалізовано перемикання між режимами:
  - `Normal` (оригінал)
  - `Inverse` (інверсія кольорів)
  - `Blur` (розмиття)
  - `Canny` (детектор границь)
- **Архітектура:** Модульна структура проєкту з розділенням на `CameraProvider`, `ObjectDetector` та `FrameProcessor`.

## Технічний звіт щодо модуля детекції
На даному етапі модуль детекції об'єктів було вимкнено бо під час спроби ініціалізації мережі через `cv::dnn::readNetFromCaffe` у поточному середовищі виконання виникає помилка `cv::Exception` (failed: `fs.is_open()`). Це пов'язано з обмеженнями доступу до файлової системи або несумісністю бібліотек OpenCV/Caffe у даній конфігурації ОС.

**Рішення:**
Для забезпечення стабільності роботи застосунку та запобігання `core dump`, виклик завантаження моделі та `net.forward()` було закоментовано. Замість динамічної детекції в метод `detect()` додано візуальну розмітку (ROI), що дозволяє продемонструвати працездатність конвеєра обробки кадрів без ризику аварійного завершення програми.
