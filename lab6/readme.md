# Lab 6 - Computer Vision Basics

Программа для обрабки відеопотока в реальному часі з використанням OpenCV.

## Функціонал
- Обработка кадрів в реальному часі:
  - Normal (без змін)
  - Invert (інверсия кольорів)
  - Blur (розмиття)
  - Canny (детектор границь)
