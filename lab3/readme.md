# Лабораторна робота №3

## Опис завдання
Завантаження, обробка та візуальний аналіз медичних даних на основі датасету heart.csv.

## Вимоги до системи
- **ОС:** Windows / Linux / macOS
- **Інтерпретатор:** Python 3.10+

## Налаштування середовища та встановлення залежностей
1. Створіть віртуальне середовище: python -m venv .venv
2. Активуйте середовище:
   - Windows: `.venv\Scripts\activate`
3. Встановіть необхідні пакети: `pip install -r requirements.txt`

## Отримання датасету
Датасет heart.csv має невеликий розмір та доданий до репозиторію.
