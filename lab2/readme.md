# Лабораторна робота №2

## Опис завдання
Проект складається з двох частин:
1. Автоматизоване завантаження, очищення та аналіз даних вегетаційного індексу (VHI) з сервера NOAA.
2. Обробка та аналіз великого набору даних споживання електроенергії (`household_power_consumption.txt`), виконання складних вибірок, масштабування даних (StandardScaler, MinMaxScaler) та One Hot Encoding.

## Вимоги до системи
- **ОС:** Windows / Linux / macOS
- **Інтерпретатор:** Python 3.10+

## Налаштування середовища та встановлення залежностей
1. Створіть віртуальне середовище:
   python -m venv .venv

2.Активуйте середовище:
    .venv\Scripts\activate

3.Встановіть необхідні пакети:
    pip install -r requirements.txt

4.Отримання датасету
    Дані VHI завантажуються автоматично під час виконання скрипту.
    Дані електроспоживання: файл household_power_consumption.txt не завантажено до репозиторію через великий розмір.
