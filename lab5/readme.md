## Опис завдання
Розробка інтерактивного веб-застосунку за допомогою фреймворку Streamlit. Застосунок призначений для аналізу, фільтрації та візуалізації часових рядів що отримані від NOAA.

## Вимоги до системи
- **ОС:** Windows / Linux / macOS
- **Інтерпретатор:** Python 3.10+

## Налаштування середовища та встановлення залежностей
1. Створіть віртуальне середовище: `python -m venv .venv`
2. Активуйте середовище:
   - Windows: `.venv\Scripts\activate`
3. Встановіть необхідні пакети: `pip install -r requirements.txt`

## Отримання датасету
Дані (CSV-файли) зчитуються з локальної теки `data/`. Ця тека не додається до репозиторію (через `.gitignore`), тому для коректної роботи скрипту переконайтеся, що ви завантажили файли VHI та поклали їх у директорію `data/` всередині `lab5`.

## Запуск проєкту
Оскільки це веб-застосунок на Streamlit, для його запуску відкрийте термінал у теці проєкту та виконайте команду:
streamlit run app.py
