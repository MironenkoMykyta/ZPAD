## Опис завдання
Генерація гармонічного сигналу з додаванням шуму та його подальша фільтрація. Побудова інтерактивного графіку за допомогою matplotlib.widgets з можливостю налаштування параметрів амплітуди, частоти, фази, характеристик шуму та частоти зрізу низькочастотного фільтра Баттерворта.

## Вимоги до системи
- **ОС:** Windows / Linux / macOS
- **Інтерпретатор:** Python 3.10+

## Налаштування середовища та встановлення залежностей
1. Створіть віртуальне середовище: `python -m venv .venv`
2. Активуйте середовище:
   - Windows: `.venv\Scripts\activate`
3. Встановіть необхідні пакети: `pip install -r requirements.txt`

## Отримання датасету
У цій лабораторній роботі зовнішні датасети не використовуються. Всі дані (чиста гармоніка та шум) генеруються програмно під час виконання скрипту.

## Запуск проєкту
Для запуску коду відкрийте Jupyter Notebook за допомогою команди: `jupyter notebook lab4.ipynb` 
Після запуску відкрийте файл у браузері та виконайте всі комірки послідовно (Run All).
